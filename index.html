Entendendo Git e GitHub

O que é Git?

Git é como uma ferramenta para guardar as versões de um projeto, ajudando a organizar tudo. Com ele, podemos voltar a versões antigas do nosso trabalho se algo der errado. É como usar o "desfazer" em um editor de texto, mas para códigos e arquivos importantes.

Também dá para ver todas as mudanças que foram feitas, quem as fez e quando. Isso ajuda muito em projetos feitos por várias pessoas ou mesmo quando você está trabalhando sozinho.

O que é GitHub?

O GitHub é um lugar na internet onde você pode guardar seus projetos feitos com o Git. É como uma nuvem especial para programadores. Assim, você não perde seus arquivos, mesmo que algo aconteça com seu computador.

Além disso, o GitHub é ótimo para trabalhar em equipe. Cada pessoa pode fazer mudanças no projeto sem bagunçar o trabalho dos outros. Para isso, usamos as "branches", que são como versões separadas do projeto. Quando tudo estiver pronto, juntamos as mudanças na versão principal.

Como começar com Git

1. Instalando o Git

Entre no site oficial do Git clicando aqui e baixe o programa. Instale seguindo as instruções do site. Quando terminar, reinicie o computador para garantir que está tudo funcionando.

Depois, abra o terminal (PowerShell no Windows ou Terminal no macOS/Linux) e digite:

git --version

Se aparecer a versão do Git, está tudo certo!

2. Criando uma conta no GitHub

Acesse o site do GitHub e crie uma conta gratuita. Você vai precisar dela para guardar seus projetos na nuvem.

3. Começando o projeto

  • No computador, escolha ou crie uma pasta para o projeto.
  • Abra o terminal dentro dessa pasta e digite:
git init

Isso começa o controle de versões do seu projeto.

Comandos básicos do Git

  • Configurar:
    • git config --global user.name "Seu Nome"
    git config --global user.email "seuemail@example.com"
  • Trabalhar no projeto:
    • git add .
    git commit -m "Mensagem"
  • Enviar para o GitHub:
    • git remote add origin URL
    git push -u origin main

Conclusão

O Git e o GitHub são ferramentas incríveis para guardar e organizar seus projetos. Eles também ajudam muito quando você trabalha em equipe. Comece instalando o Git, configure sua conta no GitHub e teste os comandos. Em pouco tempo, você vai ver como é fácil e útil usar essas ferramentas!
